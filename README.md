# Simplest online notepad for copy-paste

[https://note.initedit.com](https://note.initedit.com)

## Installation :

1. upload files to /public_html/
2. Done( Goto https://yourip/)

## Shortcuts key :

ALT  + N = New File

ALT  + C = Copy all

CTRL + S = Save

CTRL + D = Download

CTRL + L = Lock , Edit


## File status :

Locked            = Uneditable

Locked - editable = Editable (Password)

Unlocked          = Editable
